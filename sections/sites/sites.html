Desarrollo web

El desarrollo web se trata de programar sitios que serán hosteados en algún servidor y expuestos a la red para que otros usuarios puedan acceder a ellos a través de un navegador. Cuando dichos sitios son expuestos, normalmente el medio para acceder a ellos es a través de una url legible para el usuario. Sin embargo, dicha url no es más que un nombre de dominio asociado a la ip del servidor donde se encuentra hosteado el sitio, bondadosamente, el DNS (Servidor de nombres de dominio) se encarga de hacer el redireccionamiento a la ip para que el usuario no deba escribir una cadena de números en el navegador.
